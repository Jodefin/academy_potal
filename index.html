🔥 Firebase 설정 방법
1단계 — Firebase 프로젝트 만들기
① console.firebase.google.com 접속
② 프로젝트 만들기 클릭 → 이름 입력 (예: academy-dashboard)
③ Google 애널리틱스는 사용 안 함으로 설정 후 생성
2단계 — Realtime Database 생성
① 왼쪽 메뉴에서 빌드 → Realtime Database 클릭
② 데이터베이스 만들기 → 테스트 모드로 시작 (나중에 보안 규칙 변경 가능)
③ 지역은 asia-southeast1 (싱가포르) 또는 us-central1 선택
3단계 — 웹 앱 등록 및 설정값 복사
① 프로젝트 개요 → 웹 앱 추가 ( 아이콘)
② 앱 닉네임 입력 후 등록
③ firebaseConfig 객체 안의 값들을 복사
4단계 — 이 파일에 붙여넣기
이 HTML 파일 상단의 FIREBASE_CONFIG 객체를 복사한 값으로 교체하면 완료!

파일을 공유받은 모든 사람이 같은 일정·과목 데이터를 실시간으로 공유합니다.
💡 보안 규칙 (선택)
Realtime Database → 규칙 탭에서 아래처럼 설정하면
